Recordle

Rules:

  • The album resets everyday (resets at midnight local time)
  • You have three chances to guess the album correctly
  • After each wrong guess a new clue appears (left to right)
  • After a correct guess, or three subsequent wrong guesses, the actual album cover appears above
  • If u guess within 3, u win, if not u lose :)

About:

  • Clues are generated using stable diffusion, using the album title as a prompt. Model used: https://github.com/huggingface/diffusers
  • Source code: https://github.com/tg295/recordle
